Usage Guide

Learn how to use the Senchin tool in your project.

Basic Usage

After installing the tool, you can use the following code to start, pause, and reset the timer:


// Import the timer
const Timer = require('simple-timer');

// Create a new timer instance
const timer = new Timer();

// Start the timer
timer.start();

// Pause the timer
timer.pause();

// Reset the timer
timer.reset();

Timer Methods

  • start(): Begins the timer.
  • pause(): Pauses the timer.
  • reset(): Resets the timer to zero.

For more detailed documentation, check out the npm page.
